# LumosCalibView

## Project Overview
QT Application for connecting to robots and other hardware devices over various interfaces to receive data visualization and push calibration updates.

## Features
- **Multi-interface connectivity**: Supports ethernet, UART, Bluetooth, and WiFi connections
- **Data visualization**: Real-time graphing of received data
- **Calibration management**: Push updated calibration data to connected devices
- **Hardware integration**: Connect to robots and other hardware systems

## Interfaces Supported
- Ethernet
- UART
- Bluetooth  
- WiFi

## Core Functionality
1. Establish connections to hardware devices
2. Receive and visualize data in real-time graphs
3. Send calibration data updates to connected devices
4. Monitor device status and connection health
